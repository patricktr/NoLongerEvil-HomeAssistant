# No Longer Evil

Control your No Longer Evil smart thermostat from Home Assistant.

## Features

- Full climate control (temperature, mode, fan)
- Temperature and humidity sensors
- HVAC status binary sensors
- Away mode switch
- Support for heat-cool (auto) mode with temperature ranges
- Multiple thermostat support

## Requirements

- A No Longer Evil account with at least one registered thermostat
- An API key from your No Longer Evil account settings

## Setup

1. Get an API key from [No Longer Evil Settings](https://nolongerevil.com/settings)
2. Add the integration in Home Assistant
3. Enter your API key

For detailed instructions, see the [README](https://github.com/patricktr/NoLongerEvil-HomeAssistant#readme).
